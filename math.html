📌 一元二次方程求根公式

x = (-b ± √(b²-4ac)) / 2a

对于 ax² + bx + c = 0，当判别式 Δ = b²-4ac > 0 时有两个不等实根，= 0 时有两个相等实根，< 0 时无实根。

📌 韦达定理

x₁ + x₂ = -b/a, x₁·x₂ = c/a

一元二次方程两根之和与两根之积与系数的关系，常用于不求根直接求相关代数式的值。

📌 二项式定理

(a+b)ⁿ = Σ C(n,k)·aⁿ⁻ᵏ·bᵏ

展开式中第k+1项的系数为组合数C(n,k)，常用于求特定项系数或证明恒等式。

📌 三角形面积公式

S = ½ab·sinC = √[s(s-a)(s-b)(s-c)]

海伦公式中 s = (a+b+c)/2 为半周长，适用于已知三边求面积。

📌 正弦定理与余弦定理

a/sinA = b/sinB = c/sinC = 2R

c² = a² + b² - 2ab·cosC

R为外接圆半径，正弦定理用于已知两角一边求其他边，余弦定理用于已知两边夹角求第三边。

📌 圆锥曲线标准方程

椭圆: x²/a² + y²/b² = 1

双曲线: x²/a² - y²/b² = 1

抛物线: y² = 2px

📌 二次函数顶点式

y = a(x-h)² + k

顶点坐标为(h, k)，a > 0 时开口向上，a < 0 时开口向下。|a| 越大开口越窄。

📌 指数与对数运算法则

aᵐ·aⁿ = aᵐ⁺ⁿ, (aᵐ)ⁿ = aᵐⁿ

logₐ(MN) = logₐM + logₐN

logₐ(M/N) = logₐM - logₐN

📌 导数基本公式

(xⁿ)' = nxⁿ⁻¹, (eˣ)' = eˣ, (ln x)' = 1/x

(sin x)' = cos x, (cos x)' = -sin x

📌 排列与组合

A(n,m) = n!/(n-m)!

C(n,m) = n!/[m!(n-m)!]

排列讲究顺序，组合不讲究顺序。C(n,m) = C(n, n-m)。

📌 期望与方差

E(X) = Σ xᵢ·pᵢ

D(X) = E(X²) - [E(X)]²

期望反映随机变量的平均水平，方差反映离散程度。

📌 正态分布

X ~ N(μ, σ²)

约68%数据在(μ-σ, μ+σ)内，约95%在(μ-2σ, μ+2σ)内，约99.7%在(μ-3σ, μ+3σ)内。
