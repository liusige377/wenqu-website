## 🔐 portal.html - API平台

### 页面结构
```
┌────────────────────────────────────────┐
│  导航栏：首页 | 登录 | 注册 | 控制台    │
├────────────────────────────────────────┤
│  内容区域：                              │
│  ├─ 首页：平台介绍+可用模型列表           │
│  ├─ 登录：邮箱+密码表单                   │
│  ├─ 注册：姓名+邮箱+密码表单              │
│  └─ 控制台：API Key + 用量统计            │
└────────────────────────────────────────┘
```

### 后端接口
- `POST /v1/auth/register` - 注册
- `POST /v1/auth/login` - 登录
- `GET /v1/keys` - 获取API Key列表
- `GET /v1/balance` - 获取余额
- `GET /v1/billing` - 获取用量记录
- `POST /v1/recharge` - 充值
